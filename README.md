### First Web Project

이 웹서비스는 NBA를 너무 좋아하는 제가 주위 사람들한테도 소개해주고싶어서 만든 페이지입니다.

웹프로그래밍응용 수업을 들으면서 개인프로젝트를 진행했습니다.


Node.js , express, MongoDB, ejs템플릿과 여러 NPM 모듈을 사용했으며 Seed File을 통해서 DB에 초기 데이터를 저장하고 웹에 표시합니다.

NBA.COM 공식 홈페이지의 Player Page를 크롤링하여 DB에 넣고 웹에 시각적으로 표시해주는것이 목표입니다.

signin, signout, Logout 을 구현했으며 해싱하여 DB에 암호화하여 저장됩니다.

CRUD를 구현하여 팀을 수정하고 삭제 할 수 있습니다.
